# Formacja NextGen - aplikacja liderska (React + Firebase)

Krótko: projekt React + Tailwind + Firebase (Firestore) do zarządzania formacją Next Gen.
Kafelki (tematy) można przeciągać, edytować, dodawać i usuwać. Dane synchronizują się w czasie rzeczywistym dzięki Firestore.

## Szybkie uruchomienie lokalnie
1. Zainstaluj zależności: `npm install`
2. Utwórz projekt Firebase (Firestore) i skopiuj swoją konfigurację do `src/firebase.js`
3. Uruchom: `npm run dev`

## Deployment (GitHub -> Vercel / Netlify)
- Wrzucić repo na GitHub i podpiąć w Vercel/Netlify, dodać zmienne środowiskowe Firebase (opcjonalnie).
